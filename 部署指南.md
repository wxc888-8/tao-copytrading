# TAO 跟单系统 - 服务器部署指南

## 1. 安装必要软件（服务器上运行）

```bash
# 更新系统
apt update && apt upgrade -y

# 安装 Java 17
apt install openjdk-17-jdk -y
java -version

# 安装 MySQL 8.0
apt install mysql-server -y
mysql --version

# 安装 Nginx
apt install nginx -y

# 安装 Node.js 18+ (用于前端)
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install nodejs -y
node -v
npm -v
```

## 2. 配置 MySQL

```bash
mysql -u root
```

## 3. 上传项目文件

从本地电脑上传到服务器：
```bash

scp -r C:\Users\Uzi\Desktop\tao\tao-backend root@43.155.208.90:/root/

scp C:\Users\Uzi\Desktop\tao\package.json root@43.155.208.90:/root/tao-frontend/
```

## 4. 启动后端

```bash
cd /root/tao-backend
# 编译并启动
./gradlew build -x test
# 后台运行后端
nohup java -jar build/libs/tao-backend-0.0.1-SNAPSHOT.jar > app.log 2>&1 &
```

## 5. 构建前端

```bash
cd /root/tao-frontend
npm install
npm run build
# 构建产物在 dist/ 目录
```

## 6. 配置 Nginx

```bash
cat > /etc/nginx/sites-available/tao << 'EOF'
server {
    listen 80;
    server_name 43.155.208.90;

    # 前端页面
    root /root/tao-frontend/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    # 后端 API 代理
    location /api/ {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # WebSocket 代理
    location /ws/ {
        proxy_pass http://127.0.0.1:8080;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
EOF

# 启用站点
ln -sf /etc/nginx/sites-available/tao /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default
nginx -t
systemctl restart nginx
```

## 7. 验证

```bash
# 检查后端
curl http://127.0.0.1:8080/api/auth/login
# 应该返回 405 Method Not Allowed（说明后端正常）

# 检查前端
curl http://127.0.0.1:80
# 应该返回 index.html 内容
```
